# 🖥️ Server Admin Bot

Telegram-бот для адміністрування Linux-серверів (Arch Linux, Ubuntu, Debian).

## ✨ Можливості

- 🖥️ **Система** — CPU, RAM, GPU, температура, аптайм
- 💾 **Диски** — монтування/демонтування, прогрес-бари
- 🌐 **Мережа** — локальний/публічний IP, перевірка портів, SSH сесії
- ⚙️ **Сервіси** — керування systemd (start/stop/restart), кастомні сервіси
- 💻 **Консоль** — виконання команд прямо з Telegram
- 🛠️ **Команди** — створення/редагування/видалення кастомних кнопок
- ⏻ **Живлення** — перезавантаження/вимкнення сервера
- 🔐 **Авторизація** — захист паролем, прив'язка до конкретного чату

## 📋 Вимоги

- Python 3.10+
- Linux (Arch/Ubuntu/Debian)
- systemd (для керування сервісами)
- sudo права
